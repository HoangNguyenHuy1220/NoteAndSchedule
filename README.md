# NoteAndSchedule
A Mobile app for keeping notes and habit tracker using Android framework.

# Screenshot for each feature
- Text and list notes.
- Archive and recycle bin.
- Manage categories.
- Searching notes.
- Light and dark theme support.
- Arrange notes by date, alphabet.
